Simulador de descuentos

Es un simulador de compra de bolsas de cemento, cal y arena, el mismo tiene un sistema para aplicar un descuento dependiendo de la cantidad de bolsas totales compradas, Si el usuario compra mas de 10 bolsas se hace un 5% de descuento. Si el usuario compra mas de 20 bolsas se hace un 10% de descuento. Si el usuario compra mas de 30 bolsas se hace un 15% de descuento. Si el usuario compra mas de 40 bolsas se hace un 30% de descuento. la aplicacion muestra el total, luego el precio que pagara con descuento y por ultimo la cantidad que se ahorro.
